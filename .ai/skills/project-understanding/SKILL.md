Project Understanding — pepperextra

This skill documents the repository layout, apps, packages, and workspace configuration so AI agents can quickly understand how to build, run, and modify the project.

Files in this skill:

- `apps.md` — short descriptions and dev/build commands for each app.
- `packages.md` — purpose and important scripts/exports for internal packages.
- `configs.md` — workspace-level config notes (pnpm, turbo, tsconfig, env).
- `prompts.md` — recommended prompts and how agents should use these docs.
- `agent.instructions.md` — concise instructions for AI agents when working in this repo.

Primary goals:

- Give a clear mental model of the monorepo (monorepo manager, task runner, packages vs apps).
- Provide actionable commands to run, build, and test apps and packages.
- Call out important environment variables and developer conventions.

Usage: Read the module files and follow `agent.instructions.md` for how to act.
